以一张图片为例，一张彩色图片有 RGB 3个通道。但在深层神经网络中，一张图片会被提取出很多通道（比如 64 个、128 个、256 个通道特征图）。

GroupNorm 的做法是：

把这 256 个通道，平均分成几个组（Groups），比如分成 32 个组，每组 8 个通道。


然后，在这 8 个通道内部，把所有像素点的值拿来算平均值和方差，进行标准化。

self.norm = nn.GroupNorm(num_groups, out_channels)  # 如果 out_channels=256，num_groups=32，就是把 256 个通道分成 32 组，每组 8 个通道，在组内做归一化。

```python
import torch

# 1. linspace: 每一步要破坏的比例 (单步破坏力)
beta = torch.linspace(0.01, 0.03, 3) 
# 结果: [0.01, 0.02, 0.03]

# 2. 转换为每一步保留的比例
alpha = 1 - beta
# 结果: [0.99, 0.98, 0.97]

# 3. cumprod: 计算累积保留比例 (总体破坏力)
alpha_cumprod = torch.cumprod(alpha, dim=0) #alpha = [a1, a2, a3],结果是[a1, a1*a2, a1*a2*a3]
# 结果: [0.99,  0.9702,  0.941094]

```
