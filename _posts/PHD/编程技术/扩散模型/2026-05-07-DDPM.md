DDPM (Denoising Diffusion Probabilistic Models )

原理: DDPM 基于扩散过程，通过逐步向数据添加噪声，再学习如何逆转这一过程来生成数据。

过程:

前向过程: 逐步向图像添加高斯噪声，直到图像变为纯噪声。

反向过程: 通过训练神经网络逐步去噪，从纯噪声中恢复出原始图像。

特点:生成过程较慢，需要多次迭代。 生成质量高，但计算成本较大。



## DDPM详解

DDPM 是一种基于马尔可夫链(Markov Chain)的生成模型，其主要思想是：

- **前向过程（Forward Process, 加噪声）**：逐步向数据 $x_0$ 添加噪声，使其逐渐变成高斯噪声 $x_T$。
- **逆向过程（Reverse Process, 生成过程）**：从高斯噪声 $x_T$ 逐步去噪，还原出真实数据 $x_0$。
- **训练目标（Learning Objective）**：学习一个神经网络 $\epsilon_\theta(x_t, t)$ 来预测噪声，从而完成去噪过程。

# 代码已同步到github

去噪扩散概率模型（Denoising Diffusion Probabilistic Models, 简称 DDPM）是近年来生成模型领域最具突破性的工作之一。它的核心思想受到非平衡热力学的启发：**前向过程（加噪）**不断破坏数据结构，**反向过程（去噪）**学习如何从纯噪声中逐步恢复数据。
下面我将从数学推导的角度，极其详细地拆解 DDPM 的前向过程、反向过程、训练目标以及生成过程。
---
### 0. 符号定义
在开始推导前，明确所有符号：
*   $x_0$: 真实数据（如图像）。
*   $x_t$: 第 $t$ 步加噪后的数据。
*   $T$: 总的前向/反向步数（通常取 $T=1000$）。
*   $\beta_t$: 第 $t$ 步的方差，是一个超参数，满足 $0 < \beta_1 < \beta_2 < \dots < \beta_T < 1$。
*   $\alpha_t := 1 - \beta_t$。
*   $\bar{\alpha}_t := \prod_{s=1}^t \alpha_s$ （注意带杠和不带杠的区别）。
*   $\mathcal{N}(x; \mu, \Sigma)$: 均值为 $\mu$，协方差为 $\Sigma$ 的多元高斯分布。
*   $\epsilon \sim \mathcal{N}(0, I)$: 标准高斯白噪声。
---
### 1. 前向过程
前向过程是一个马尔可夫链，即每一步的状态只依赖于前一步。它的作用是**逐步向图像中加入高斯噪声，直到图像变成纯噪声**。
#### 1.1 单步加噪
根据马尔可夫性质，第 $t$ 步的加噪过程定义为：
$$q(x_t \mid x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t \mathbf{I})$$

利用重参数化技巧，可以写成：
(重参数技巧）

$$x_t = \sqrt{1 - \beta_t} x_{t-1} + \sqrt{\beta_t} \epsilon, \quad \epsilon \sim \mathcal{N}(0, \mathbf{I})$$
#### 1.2 任意时刻 $t$ 的加噪公式（核心推导 1）
在训练时，如果每次都要从 $x_0$ 一步一步计算到 $x_t$，效率太低。我们需要推导出**直接从 $x_0$ 得到 $x_t$ 的解析解**。
利用方差的高斯分布相加的性质：
$$x_t = \sqrt{\alpha_t} x_{t-1} + \sqrt{1 - \alpha_t} \epsilon_{t-1}$$
将 $x_{t-1}$ 展开：
$$x_t = \sqrt{\alpha_t} (\sqrt{\alpha_{t-1}} x_{t-2} + \sqrt{1 - \alpha_{t-1}} \epsilon_{t-2}) + \sqrt{1 - \alpha_t} \epsilon_{t-1}$$
$$x_t = \sqrt{\alpha_t \alpha_{t-1}} x_{t-2} + \sqrt{\alpha_t(1 - \alpha_{t-1})} \epsilon_{t-2} + \sqrt{1 - \alpha_t} \epsilon_{t-1}$$
由于两个独立高斯随机变量的线性组合仍然是高斯分布：
令 $a = \sqrt{\alpha_t(1 - \alpha_{t-1})}$, $b = \sqrt{1 - \alpha_t}$，那么组合后的方差为 $a^2 + b^2 = \alpha_t - \alpha_t\alpha_{t-1} + 1 - \alpha_t = 1 - \alpha_t\alpha_{t-1}$
所以我们可以把噪声项合并为一个新的标准高斯噪声 $\epsilon \sim \mathcal{N}(0, \mathbf{I})$，继续递归下去：
$$x_t = \sqrt{\alpha_t \alpha_{t-1} \dots \alpha_1} x_0 + \sqrt{1 - \alpha_t \alpha_{t-1} \dots \alpha_1} \epsilon$$
代入 $\bar{\alpha}_t$ 的定义，得到极其重要的**一步加噪公式**：
$$q(x_t \mid x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t} x_0, (1 - \bar{\alpha}_t)\mathbf{I})$$
或者写成采样形式：
$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon$$

**意义：** 这个公式说明，我们可以直接给原始图像 $x_0$ 乘上一个缩放系数，再加上特定比例的噪声，就能得到任意时刻 $t$ 的噪声图像。这是 DDPM 训练极其高效的根源。

### 2. 反向过程
反向过程的目标是**从纯噪声 $x_T$ 开始，逐步去噪，最终恢复出真实图像 $x_0$**
#### 2.1 真实的反向分布
如果我们知道真实的后验分布 $q(x_{t-1} \mid x_t)$，我们就能完美去噪。根据贝叶斯公式：
$$q(x_{t-1} \mid x_t) = \frac{q(x_t \mid x_{t-1}) q(x_{t-1})}{q(x_t)}$$
但 $q(x_{t-1})$ 和 $q(x_t)$ 是整个数据集的分布，**难以求解**。

#### 2.2 真实后验的解析形式（核心推导 2）
虽然无法直接求，但我们可以利用前向过程的马尔可夫性，求出以 $x_0$ 为条件的后验 $q(x_{t-1} \mid x_t, x_0)$。
因为 $x_0 \to x_{t-1} \to x_t$ 是马尔可夫链，所以：
$$q(x_{t-1} \mid x_t, x_0) = q(x_t \mid x_{t-1}, x_0) \frac{q(x_{t-1} \mid x_0)}{q(x_t \mid x_0)} = q(x_t \mid x_{t-1}) \frac{q(x_{t-1} \mid x_0)}{q(x_t \mid x_0)}$$
注意等式右边全部都是**高斯分布**！两个高斯分布相乘除，结果仍然是高斯分布。

*   $q(x_t \mid x_{t-1}) = \mathcal{N}(x_t; \sqrt{\alpha_t}x_{t-1}, \beta_t \mathbf{I})$
*   $q(x_{t-1} \mid x_0) = \mathcal{N}(x_{t-1}; \sqrt{\bar{\alpha}_{t-1}}x_0, (1-\bar{\alpha}_{t-1})\mathbf{I})$
*   $q(x_t \mid x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t}x_0, (1-\bar{\alpha}_t)\mathbf{I})$

利用高斯分布的指数相乘（配方法），求解均值 $\tilde{\mu}_t$ 和方差 $\tilde{\beta}_t$（具体代数展开较繁琐，此处直接给出严谨结果）：
**方差 $\tilde{\beta}_t$：**
$$\tilde{\beta}_t = \frac{(1 - \bar{\alpha}_{t-1}) \beta_t}{1 - \bar{\alpha}_t}$$
**均值 $\tilde{\mu}_t$：**
$$\tilde{\mu}_t = \frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t}{1 - \bar{\alpha}_t} x_0 + \frac{\sqrt{\alpha_t} (1 - \bar{\alpha}_{t-1})}{1 - \bar{\alpha}_t} x_t$$

#### 2.3 用噪声 $\epsilon$ 表示真实均值（核心推导 3）
我们注意到 $\tilde{\mu}_t$ 里面包含了 $x_0$，但去噪时我们手里只有 $x_t$，没有 $x_0$。怎么办？
利用前向过程的公式 $x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon$，反解出 $x_0$：
$$x_0 = \frac{1}{\sqrt{\bar{\alpha}_t}} (x_t - \sqrt{1 - \bar{\alpha}_t} \epsilon)$$
将这个 $x_0$ 代入到 $\tilde{\mu}_t$ 中：
$$\tilde{\mu}_t = \frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t}{1 - \bar{\alpha}_t} \left[ \frac{1}{\sqrt{\bar{\alpha}_t}} (x_t - \sqrt{1 - \bar{\alpha}_t} \epsilon) \right] + \frac{\sqrt{\alpha_t} (1 - \bar{\alpha}_{t-1})}{1 - \bar{\alpha}_t} x_t$$
提取 $x_t$ 的系数：
系数 = $\frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t}{\sqrt{\bar{\alpha}_t}(1 - \bar{\alpha}_t)} + \frac{\sqrt{\alpha_t} (1 - \bar{\alpha}_{t-1})}{1 - \bar{\alpha}_t}$
因为 $\bar{\alpha}_t = \bar{\alpha}_{t-1}\alpha_t$，所以 $\sqrt{\bar{\alpha}_{t-1}} = \frac{\sqrt{\bar{\alpha}_t}}{\sqrt{\alpha_t}}$。代入后通分，分子会极其巧妙地消去：
$\frac{\beta_t}{\sqrt{\alpha_t}(1-\bar{\alpha}_t)} + \frac{\sqrt{\alpha_t}(1-\bar{\alpha}_{t-1})}{1-\bar{\alpha}_t} = \frac{\beta_t + \alpha_t(1-\bar{\alpha}_{t-1})}{\sqrt{\alpha_t}(1-\bar{\alpha}_t)}$
由于 $\beta_t = 1 - \alpha_t$，分子变为 $1 - \alpha_t + \alpha_t - \alpha_t\bar{\alpha}_{t-1} = 1 - \bar{\alpha}_t$。
所以 $x_t$ 的系数化简为：$\frac{1}{\sqrt{\alpha_t}}$。
提取 $\epsilon$ 的系数：
$$-\frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t \sqrt{1 - \bar{\alpha}_t}}{\sqrt{\bar{\alpha}_t}(1 - \bar{\alpha}_t)} = -\frac{\beta_t}{\sqrt{\alpha_t} \sqrt{1 - \bar{\alpha}_t}}$$
最终得到非常优美的均值公式：
$$\tilde{\mu}_t = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon \right)$$
**意义：** 这个公式告诉我们，只要我们知道了第 $t$ 步加入的噪声 $\epsilon$，我们就能算出完美的去噪均值，从而得到 $x_{t-1}$。

#### 2.4 参数化近似反向分布
既然真实的反向分布 $q(x_{t-1} \mid x_t)$ 求不出来，我们就**用神经网络去拟合它**。
设神经网络参数为 $\theta$，定义近似分布：
$$p_\theta(x_{t-1} \mid x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$
DDPM 原论文中做了一个简化假设：**方差固定为前向过程的方差**（后续改进版如 Improved DDPM 将方差也设为可学习的，这里先按原版推导）：
$$\Sigma_\theta(x_t, t) = \beta_t \mathbf{I}$$
对于均值 $\mu_\theta$，根据前面的推导，它应该长成 $\tilde{\mu}_t$ 那样。因此我们让神经网络去**预测噪声 $\epsilon$**：
$$\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)$$

其中 $\epsilon_\theta(x_t, t)$ 就是一个 U-Net 神经网络（输入 $x_t$ 和时间步 $t$，输出与 $x_t$ 同尺寸的预测噪声）。

### 3. 训练过程：变分下界（ELBO推导）
我们要训练参数 $\theta$，目标是最大化数据的对数似然 $\log p_\theta(x_0)$。由于这个似然无法直接求，我们最大化它的**变分下界**。
#### 3.1 ELBO 的展开
利用变分推断的标准套路，加上并减去 $q(x_{1:T} \mid x_0)$：
$$\log p_\theta(x_0) = \log \int p_\theta(x_{0:T}) dx_{1:T} = \log \int \frac{p_\theta(x_{0:T})}{q(x_{1:T} \mid x_0)} q(x_{1:T} \mid x_0) dx_{1:T}$$
利用 Jensen 不等式 $\log(\mathbb{E}[X]) \ge \mathbb{E}[\log(X)]$：
$$\log p_\theta(x_0) \ge \mathbb{E}_{q(x_{1:T} \mid x_0)} \left[ \log \frac{p_\theta(x_{0:T})}{q(x_{1:T} \mid x_0)} \right] := -L_{VLB}$$
将分子分母展开为连乘：
*   $p_\theta(x_{0:T}) = p(x_T) \prod_{t=1}^T p_\theta(x_{t-1} \mid x_t)$
*   $q(x_{1:T} \mid x_0) = \prod_{t=1}^T q(x_t \mid x_{t-1})$
    取对数后相减：
    $$-L_{VLB} = \mathbb{E}_q \left[ \log p(x_T) - \sum_{t=1}^T \log \frac{q(x_t \mid x_{t-1})}{p_\theta(x_{t-1} \mid x_t)} \right]$$
    此时，运用一个极其关键的技巧：**在求和号内部，乘上并除以 $q(x_{t-1} \mid x_t, x_0)$**：
    $$-L_{VLB} = \mathbb{E}_q \left[ \log p(x_T) - \sum_{t=1}^T \log \frac{q(x_t \mid x_{t-1})}{p_\theta(x_{t-1} \mid x_t)} \cdot \frac{q(x_{t-1} \mid x_t, x_0)}{q(x_{t-1} \mid x_t, x_0)} \right]$$
    重新组合，将无关项提取出去：
    $$-L_{VLB} = \mathbb{E}_q \left[ \log \frac{p(x_T)}{q(x_T \mid x_0)} + \sum_{t=2}^T \log \frac{q(x_{t-1} \mid x_t, x_0)}{p_\theta(x_{t-1} \mid x_t)} - \log p_\theta(x_0 \mid x_1) \right]$$
    两边乘以 -1，得到损失函数 $L_{VLB}$ 的分解：
    $$L_{VLB} = \underbrace{\mathbb{E}_q \left[ D_{KL}(q(x_T \mid x_0) \| p(x_T)) \right]}_{L_T} + \sum_{t=2}^T \underbrace{\mathbb{E}_q \left[ D_{KL}(q(x_{t-1} \mid x_t, x_0) \| p_\theta(x_{t-1} \mid x_t)) \right]}_{L_{t-1}} \underbrace{- \mathbb{E}_q[\log p_\theta(x_0 \mid x_1)]}_{L_0}$$
*   $L_T$: 先验匹配项，没有可训练参数，可以忽略。
*   $L_0$: 解码项，可以用一个离散化高斯分布处理，原论文中为了简化，将其统一到了 $L_{t-1}$ 中。
*   **$L_{t-1}$: 这是训练的核心！**
#### 3.2 化简 $L_{t-1}$（核心推导 4）
$L_{t-1}$ 是两个高斯分布的 KL 散度。对于两个多元高斯分布 $\mathcal{N}(\mu_1, \Sigma_1)$ 和 $\mathcal{N}(\mu_2, \Sigma_2)$，KL散度为：
$$D_{KL} = \frac{1}{2} \left[ \log \frac{|\Sigma_2|}{|\Sigma_1|} - d + \text{tr}(\Sigma_2^{-1}\Sigma_1) + (\mu_2 - \mu_1)^T \Sigma_2^{-1} (\mu_2 - \mu_1) \right]$$
在我们的场景中：
*   真实后验：$\mu_1 = \tilde{\mu}_t$, $\Sigma_1 = \tilde{\beta}_t \mathbf{I}$
*   神经网络预测：$\mu_2 = \mu_\theta(x_t, t)$, $\Sigma_2 = \beta_t \mathbf{I}$ (原论文假设)
    因为都是对角协方差矩阵，KL 散度简化为：
    $$L_{t-1} = \mathbb{E}_q \left[ \frac{1}{2\beta_t} \|\tilde{\mu}_t - \mu_\theta(x_t, t)\|^2 \right] + C$$
    （$C$ 是与 $\theta$ 无关的常数，可忽略）。
    把前面推导的 $\tilde{\mu}_t$ 和 $\mu_\theta$ 代入：
    $$\|\tilde{\mu}_t - \mu_\theta\|^2 = \left\| \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon \right) - \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right) \right\|^2$$
    提出公共项 $\frac{\beta_t}{\sqrt{\alpha_t}\sqrt{1-\bar{\alpha}_t}}$：
    $$\|\tilde{\mu}_t - \mu_\theta\|^2 = \left( \frac{\beta_t}{\sqrt{\alpha_t}\sqrt{1-\bar{\alpha}_t}} \right)^2 \|\epsilon - \epsilon_\theta(x_t, t)\|^2$$
    所以 $L_{t-1}$ 变为：
    $$L_{t-1} = \mathbb{E}_{x_0, \epsilon} \left[ \frac{\beta_t^2}{2 \alpha_t (1-\bar{\alpha}_t) \beta_t} \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right] = \mathbb{E} \left[ \frac{\beta_t}{2 \alpha_t (1-\bar{\alpha}_t)} \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]$$
#### 3.3 最终的简化损失函数（Ho et al. 的神来之笔）
严格来说，应该按照上面带权重的公式优化。但是 Ho 等人在论文中发现了一个极其有效且简单的经验结论：**把前面的权重项全部去掉，效果反而更好，训练也更稳定**。
此外，将 $t=1$ 的情况也用类似形式包含进来。于是得到了 DDPM 最经典的训练目标：
$$L_{simple}(\theta) = \mathbb{E}_{t, x_0, \epsilon} \left[ \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]$$
其中：
1.  $t \sim \text{Uniform}(1, \dots, T)$：随机采样一个时间步。
2.  $x_0 \sim q(x_0)$：从数据集中取一张真实图片。
3.  $\epsilon \sim \mathcal{N}(0, \mathbf{I})$：生成纯随机噪声。
4.  计算 $x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon$。
5.  优化预测噪声 $\epsilon_\theta$ 与真实噪声 $\epsilon$ 之间的**均方误差（MSE）**。
---
### 4. 生成过程
训练好网络 $\epsilon_\theta$ 之后，我们如何生成图像呢？这就是反向过程的采样。
我们从纯高斯噪声 $x_T \sim \mathcal{N}(0, \mathbf{I})$ 开始，逐步循环 $t = T, T-1, \dots, 1$。
根据我们定义的近似后验分布：
$$p_\theta(x_{t-1} \mid x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \beta_t \mathbf{I})$$
利用重参数化技巧进行采样：
$$x_{t-1} = \mu_\theta(x_t, t) + \sqrt{\beta_t} z, \quad z \sim \mathcal{N}(0, \mathbf{I}) \text{ (当 } t > 1 \text{ 时)}$$
$$x_{t-1} = \mu_\theta(x_t, t) \quad \text{ (当 } t = 1 \text{ 时，不加噪声)}$$
将 $\mu_\theta(x_t, t)$ 的具体形式代入，得到最终用于代码编写的**采样公式**：
$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right) + \sqrt{\beta_t} z$$
**生成过程的直观理解：**

在每一步 $t$，网络观察当前的噪声图像 $x_t$，预测出其中包含的噪声 $\epsilon_\theta$。然后我们从 $x_t$ 中**减去**这部分预测出的噪声（经过适当的权重缩放），再加上一点微小的随机扰动 $\sqrt{\beta_t} z$（为了保证马尔可夫链的随机性，避免陷入确定性路径的病态解），就得到了稍微清晰一点的图像 $x_{t-1}$。重复1000次，即可得到高清图像。

### 5. 总结：从数学到代码的映射
如果你要写一个最简单的 DDPM，代码的核心逻辑完全对应上述数学推导：
#### 训练循环:
```python
# 1. 取真实图像
x_0 = next(dataloader)
# 2. 随机采样时间步 t 和 噪声 epsilon
t = torch.randint(0, T, (batch_size,))
epsilon = torch.randn_like(x_0)
# 3. 对应前向过程推导：一步加噪得到 x_t
alpha_t = alphas[t]
alpha_bar_t = alpha_cumprod[t]
x_t = torch.sqrt(alpha_bar_t) * x_0 + torch.sqrt(1 - alpha_bar_t) * epsilon
# 4. 送入网络，预测噪声
epsilon_pred = unet(x_t, t)
# 5. 对应 L_simple：计算 MSE Loss
loss = mse_loss(epsilon, epsilon_pred)
loss.backward()
```
#### 生成循环:
```python
# 1. 从纯噪声开始
x_t = torch.randn(image_shape)
# 2. 反向循环
for t in reversed(range(T)):
    # 网络预测噪声
    epsilon_pred = unet(x_t, t)
    
    # 对应生成公式中的均值部分
    alpha_t = alphas[t]
    alpha_bar_t = alpha_cumprod[t]
    mean = (1 / torch.sqrt(alpha_t)) * (x_t - (beta_t / torch.sqrt(1 - alpha_bar_t)) * epsilon_pred)
    
    # 对应生成公式中的随机扰动
    if t > 0:
        noise = torch.randn_like(x_t)
        x_t = mean + torch.sqrt(beta_t) * noise
    else:
        x_t = mean # 最后一步不加噪声

```

# 公式
## 重参数技巧
**核心本质**：将随机采样转化为确定性的代数运算，使梯度能够穿过采样节点。
**通用公式**：
将 $x \sim \mathcal{N}(\mu, \sigma^2)$
转换为：$x = \mu + \sigma \cdot \epsilon$
（其中 $\epsilon \sim \mathcal{N}(0,1)$ 被视为与网络参数无关的外部输入）
此时 $\frac{\partial x}{\partial \mu} = 1$，梯度链打通。
**在 DDPM 中的直接体现**：

$q(x_t \mid x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t \mathbf{I})$

1. **前向加噪**：
   $$x_t = \underbrace{\sqrt{\bar{\alpha}_t} x_0}_{\text{确定性平移}} + \underbrace{\sqrt{1 - \bar{\alpha}_t}}_{\text{确定性缩放}} \cdot \underbrace{\epsilon}_{\text{外提随机源}}$$
2. **反向去噪**：
   $$x_{t-1} = \underbrace{\mu_\theta(x_t, t)}_{\text{网络确定性输出}} + \underbrace{\sqrt{\beta_t}}_{\text{确定性缩放}} \cdot \underbrace{z}_{\text{外提随机源}}$$

虽然我们在训练时，$x_0$ 是固定的真实图片，不直接需要对 $\beta$ 求导，但我们需要把 $x_t$ 作为**神经网络的输入**。通过重参数化，我们用 `torch.randn` 生成 $\epsilon$，然后通过矩阵乘法算出 $x_t$。这个 $x_t$ 是通过确定性的张量运算得到的，它完美地融入了 PyTorch 的计算图，使得后续从 Loss 到网络参数 $\theta$ 的梯度回传成为可能。

以一维高斯分布为例，我们要从 $\mathcal{N}(\mu, \sigma^2)$ 中采样 $x$。
**原始写法（不可导）：**
$$ x \sim \mathcal{N}(\mu, \sigma^2) $$
**重参数化写法（可导）：**

1.  先从一个**固定的、与网络参数无关的**标准正态分布中采样一个基础噪声：$\epsilon \sim \mathcal{N}(0, 1)$
2.  通过一个**确定性的仿射变换**，把 $\epsilon$ “拉伸”和“平移”成我们想要的 $x$：
    $$ x = \mu + \sigma \cdot \epsilon $$
    **为什么这样就能求导了？**
    你看这个公式 $x = \mu + \sigma \cdot \epsilon$，它变成了一个普通的加减乘除运算！
    现在 $\epsilon$ 被当作是网络的一个**外部输入**（就像输入图片一样），它不参与参数更新。
    此时求导：
    $$ \frac{\partial x}{\partial \mu} = 1 $$
    $$ \frac{\partial x}{\partial \sigma} = \epsilon $$
    梯度畅通无阻！这就是重参数化的精髓
