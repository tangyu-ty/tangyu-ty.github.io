docker很麻烦，要一直启动，不然每次都要下载ide，我这个没弄明白，思来想去还是在windows上装一套




参考

[https://jekyllrb.com/docs/plugins/your-first-plugin/](https://jekyllrb.com/docs/plugins/your-first-plugin/)

[https://github.com/jekyll/jekyll-compose](https://github.com/jekyll/jekyll-compose)

[https://jekyllrb.com/docs/installation/windows/](https://jekyllrb.com/docs/installation/windows/)

rubyinstaller下载，贼慢

[https://rubyinstaller.org/downloads/](https://rubyinstaller.org/downloads/)




最后

运行

```shell
gem install jekyll bundler

gem update --system 4.0.2

```

到项目路径下，确保有`gemfile`文件

执行

```shell
bundle install
```
